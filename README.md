# Проект: Путешествие по России

Проект на закрепление навыков создания адаптивного web-приложения, комфортно просматриваемых  с разных устройств, от смартфона до большого монитора.

## Используемые технологии:
1. Методология БЭМ.
2. Git.
3. Grid.
4. Flex-box.
